Women Safety Alert System

Project Overview

The Women Safety Alert System is an Arduino-based emergency security device designed to help women during dangerous situations. When the emergency button is pressed, the system activates a buzzer alarm and sends an alert message with live GPS location to a predefined mobile number using the GSM module.

Components Used

- Arduino Nano
- SIM800L GSM Module
- NEO-6M GPS Module
- Push Button
- Buzzer
- Battery Supply
- Connecting Wires

Working Principle

1. The user presses the emergency push button.
2. Arduino Nano detects the button press.
3. The buzzer turns ON to indicate alert activation.
4. Arduino reads the current GPS coordinates from the NEO-6M GPS module.
5. The SIM800L GSM module sends an emergency SMS containing the Google Maps location link to the registered phone number.
6. The receiver can open the location directly in Google Maps and help the user quickly.

Features

- One-click emergency alert
- Real-time GPS location sharing
- SMS alert using GSM network
- Portable and low-cost design
- Easy to use during emergencies

Applications

- Women safety and protection
- Personal emergency alert system
- Student safety projects
- Security and tracking applications

Future Improvements

- Mobile app integration
- Voice recording feature
- Rechargeable battery system
- Mini wearable design

Conclusion

This project provides a simple and effective safety solution using Arduino, GSM, and GPS technologies. It helps improve personal safety by sending instant emergency alerts and live location details during critical situations.
